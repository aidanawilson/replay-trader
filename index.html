Session

Timeframe is controlled from the chart toolbar. Every scenario uses about 24 hours of lookback on the selected timeframe, and the continuous data sampler can choose from anywhere in the loaded NQ/ES data.

Data external continuous futures data ready

This build loads continuous NQ1 data for MNQ/NQ and continuous ES1 data for MES/ES from CSV files in the /data folder. You can still import another CSV if you want.
